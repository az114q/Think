# Iconfont 规范

规范旨在统一管理icon,形成标准,是图标之间保持一致的视觉效果.

## 图标命名
为了给图标赋予语义化的命名,命名规则如下:

- 实心和描线图标保持同名. 用 `-o`来区分,比如 ￼`xfont-question-circle-o`(描线) 和 `xfont-question-circle`(实心);
- 命名顺序: `xfont-[图标名]-[形状?]-[描线(o)?]`. (注: `?` 为可选,命名尽量简化);
- 命名不要耦合业务;
